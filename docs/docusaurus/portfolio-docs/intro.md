# Nick2bad4u Projects

This site turns my copied Docusaurus template into a portfolio and project explorer for everything public on my GitHub profile.

## What you can do here

- Browse a curated set of **featured repositories** on the homepage.
- Jump straight into a dedicated **ESLint plugins** collection instead of finding each plugin one by one.
- Open the full **Projects** explorer to search, sort, and filter repositories.
- Browse grouped **project collections** for apps, PowerShell work, automation, developer tooling, cycling, and gaming projects.
- Use the docs in this section to understand the kinds of work I publish and how the site is organized.

## What the projects page includes

The main project directory shows:

- repository descriptions,
- stars and forks,
- homepage or package links when available,
- language and category badges,
- topic tags pulled from GitHub metadata.

## Data source

The portfolio ships with a local repository snapshot and then tries to refresh from the public GitHub API in the browser. That means the site still works if the API is unavailable, while normally staying current without any manual editing.
