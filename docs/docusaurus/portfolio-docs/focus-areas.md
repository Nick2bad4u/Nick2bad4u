# Focus Areas

My repositories are spread across a few recurring themes.

## TypeScript and lint tooling

A big chunk of the portfolio is focused on developer tooling:

- a grouped family of ESLint plugins and experiments
- TSDoc and comment quality tooling
- Stylelint and configuration inspection utilities
- formatter and editor-adjacent tooling

## Apps and dashboards

I also build app-style projects with a strong UI angle, including:

- file viewers,
- monitoring dashboards,
- repo-backed websites,
- live demos deployed with GitHub Pages or custom domains.

## PowerShell and terminal customization

A lot of my automation and personalization work lives in PowerShell:

- prompt themes,
- terminal art and colors,
- scripts for productivity and automation,
- cross-platform shell improvements.

## Fitness, gaming, and niche utilities

Some projects are more specialized and hobby-driven:

- cycling and Zwift tools,
- FIT file viewers,
- RuneLite and game-related utilities,
- one-off experiments that solve a very specific problem.

## Why the site is structured this way

Instead of a flat list of repository names, the portfolio groups things by recognizable work patterns so it is easier to understand what I actually like building.
